# Chess Analyzer

A simple tool to track and analyze my chess games.
I play with a rating 1400, and this program helps me understand my perfomance.

## Features
- Stories game date, opponent rating, result (win/loss),
- Calculates total wins and win tate(%)
- Shows how well I play with white vs black pieces
- Built with Python, Pandas, and SQLite

## How to run
1. Open 'chess_analyzer.py'
2. Run it in Python or Google Colab
3. See your chess stats instantly!
